# Brand_perception_Dashboard
1. Project Title
Brand Perception on Service & Support for Health Care Equipments

2. Purpose
This Power BI project focuses on Brand Perception on Service & Back-Up Support for Imaging, Ultrasound, and Patient Care Solutions equipment. The study evaluates how different service providers (OEMs and third-party vendors) are perceived across critical service dimensions and how these perceptions influence overall brand satisfaction.

3. Tech Stack
The Dashboard was built using the following tools and technologies.
Power Bi Desktop - main data visualization platform for report creation.
power Query - Data transformation and cleaning layer for reshaping and preparing the data.
DAX - used for calculated measures, dynamic visuals, and conditional logic.
Data modeling - Relationships established amonng tables to enable cross filtering and aggregation.

4. Data Source
Since this was built using survey data, we get the data in the spss format which then was converted into excel for easy use.

5. Highlights
Service Responsiveness (call response time, remote support effectiveness)
Repair Turnaround Performance (remote and on-site)
Replacement Parts Availability & Quality
Training & Technical Support Quality
Value-Added & Digital Service Capabilities
Pricing & Commercial Flexibility
Overall Satisfaction & Recommendation Drivers
